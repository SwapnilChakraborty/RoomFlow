# RoomFlow - QR-Powered Hotel Management System

A premium SaaS platform that replaces in-room phones with a modern mobile interface for guests and a comprehensive dashboard for hotel operations.

## Technology Stack
- **Frontend**: React (Vite), Tailwind CSS 4, Lucide Icons, Framer Motion
- **Backend**: Node.js, Express, Socket.io, MongoDB (Mongoose)

## Getting Started

### Client
```bash
cd client
npm install
npm run dev
```

### Server
```bash
cd server
npm install
npm run dev
```

## UI Structure
The project is divided into three main interfaces:
1. **Guest**: Accessible via QR scan. Focuses on speed and simplicity.
2. **Admin**: Desktop dashboard for managers to monitor the entire property.
3. **Staff**: Mobile task manager for housekeeping and kitchen staff.
